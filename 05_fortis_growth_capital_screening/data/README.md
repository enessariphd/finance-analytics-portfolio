# Data Notes

This project is a screening-level renewable energy growth capital case study.

Raw source files are not included in this repository. The Python workflow uses transparent memo-level public-data assumptions to reproduce project economics, sensitivity tables and screening outputs.

The case is based on public information and illustrative project-economics assumptions, including:

- Tokça SPP benchmark project size and generation assumptions
- Public investment cost benchmark
- Average PTF electricity price assumption
- Operating cost / EBITDA margin assumptions
- Project IRR and payback sensitivity logic
- Self-consumption upside sensitivity

This is a screening-level analytical case study. It is not full due diligence, not a formal valuation, and not investment advice.
