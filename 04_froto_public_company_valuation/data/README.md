# Data Notes

This project is a public company valuation case study for Ford Otosan (FROTO).

Raw source files and Excel model workbooks are not included in this repository.

The Python workflow expects a locally stored final valuation workbook containing cleaned valuation outputs and sensitivity tables. The supporting source materials used in the original case include:

- Ford Otosan annual reports and financial statements
- Ford Otosan 1Q26 and 2025YE earnings releases
- BIST market price data
- Selected automotive peer financial statements and annual reports
- Public market and macro/WACC reference data
- External broker target price used only as a sanity check

The repository includes scripts, generated output tables/charts and the final valuation memo, but excludes source PDFs, raw files and Excel model versions.

This is an analytical valuation case study and is not investment advice.
